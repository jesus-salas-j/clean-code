## Dudas que salieron durante la sesión:

- Bloques if...else, como pongo las llaves, depende del lenguaje, cada uno tiene su convención.

En Java o Kotlin:
```
if (...) {
  ...
} else {
  ...
}
```

En C#:
```
if (...) {   
  ...   
} 
else 
{
  ...
}
```
